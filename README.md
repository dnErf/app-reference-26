# app-reference-26

A collection of reference implementations for modern web and database technologies.

## Projects

### Mojo Grizzly
A high-performance columnar database built in pure Mojo, inspired by DuckDB and ClickHouse. Features include:
- Columnar storage with Apache Arrow
- SQL queries with JOINs, aggregates, and advanced analytics
- Multiple data formats (Parquet, AVRO, ORC, JSONL, CSV)
- Extensions for blockchain, graph, ML, analytics, and more
- REST API for remote queries
- Time travel and versioning in lakehouse mode
- Comprehensive benchmarking suite- All planned features implemented and production-ready.
See [mojo-grizzly/README.md](mojo-grizzly/README.md) for details.

### Astro Hono React Waitlist
A full-stack waitlist application using Astro, Hono, React, and Cloudflare.
